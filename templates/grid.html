Hi translator and welcome to Polish, a user-friendly translation tool!


Basic flow

1. In the top menu there is a list of languages. Click on the one that you would like to translate sentences to.

2. What you see now is a list of the untranslated phrases for the selected language. The left column contains the phrases also called "keys" in the default language which is Swedish in our case. The right column contains the phrases that should be translated to the selected language.

3. To translate a sentence, click on the text in the right side and a text area will appear.

4. Translate the text to the selected language and then click Save translation. If you do not want to save the translation, you can click Cancel. If you think that the translation of this sentence is the same as in the default language, click Mark always as translated and the system will consider this sentence as translated forever.

5. In order for the new translations to take effect you need to submit your translations. You can submit them by clicking Submit translations in the top menu. A list of all the changes that you have done will be shown and you can revise them if you need to. Once you are confident with your translations you can click the Submit button to submit your translations.



Search

Polish also allows you to search for words or parts of words. You can click Search in the top menu. A simple form will appear where you can select your searching criteria.



Status

Polish allows multiple translators to work at the same time in the same language. You are able to see who is currently translating by clicking Status in the top menu. The status panel also contains information about the selected language, such as the total number of phrases, the total number of untranslated phrases, and also who is currently translating the same language, and how many unsubmitted phrases they have.



That's it! Pretty easy right? You can now start using Polish and translate sentences in a cool and easy way! :-)
